These pages show static-ring in action.

static-ring is a webring system that doesn't need a server to work. It can be hosted anywhere that supports html.

It uses the html tag <meta http-equiv="Refresh" ...> and javascript to redirect to the next page. A fallback link is present in case it doesn't work.

It's built with eleventy and it's fairly simple. You add the sites to the sites.js file and eleventy takes care of the rest.

You can find the source code and instructions at https://github.com/umaruru/static-ring .
